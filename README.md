
## 🚀 1. Cài đặt

### Yêu cầu:

- Node.js >= 16
- npm hoặc yarn

### Bước cài đặt:

```bash
# Tải mã nguồn
git clone https://github.com/DangHieu19022004/123kidmath.git
cd 123kidmath

# Cài đặt thư viện
npm install
# hoặc
yarn install


# Chạy
npm run dev
# hoặc
yarn dev
```
